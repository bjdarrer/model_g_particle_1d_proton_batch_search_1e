It is built around the logic we discussed: use the neutral/neutron-like branch to anchor the length scale 
via the Compton wavelength, use the proton-like branch to calibrate charge, and only then apply the
Brown/LaViolette (M/Q) ratio to get an active-mass scale. That matches the book’s use of the Turing 
wavelength as the particle Compton wavelength, its treatment of (Q_y)/(\rho_y) as the active-charge side, 
(S_g) as active gravitational mass, and the proton-like state as a secondary bifurcation off a neutral branch.

The module gives you three main entry points:

* `score_neutron_branch(...)`
* `score_proton_branch(...)`
* `combine_neutron_proton_scores(...)`

It expects your solver output in a simple dict like:

```python
sim = {
    "r": r,
    "G": G,
    "X": X,
    "Y": Y,
    "G0": G0,
    "X0": X0,
    "Y0": Y0,
}
```

and your parameter set as:

```python
params = ModelGParams(
    A=A, B=B, Dg=Dg, Dx=Dx, Dy=Dy,
    k1=k1, k2=k2, km2=km2, k3=k3, k4=k4, k5=k5,
)
```

```python

    ModelGParams,
    score_neutron_branch,

    combine_neutron_proton_scores,
)

params = ModelGParams(
    A=A, B=B, Dg=Dg, Dx=Dx, Dy=Dy,
    k1=k1, k2=k2, km2=km2, k3=k3, k4=k4, k5=k5,

score_n = score_neutron_branch(neutron_sim, params)
score_p = score_proton_branch(
```

## 5) Run the proton refinement search

If you have a previous proton winner JSON, use:

```bash
python3 model_g_particle_1d_proton_batch_search_1e.py \
  --outdir proton_refine_run_1e \
  --refine-from /path/to/previous/best_candidate.json \
  --neutron-anchor-json neutron_anchor_run_1a/best_candidate_neutron_anchor.json \
  --rank-by combined \
  --sqk-weight 0.35
```

If your earlier result is only in `summary.txt`, use:

```bash
python3 model_g_particle_1d_proton_batch_search_1e.py \
  --outdir proton_refine_run_1e \
  --refine-from /path/to/previous/summary.txt \
  --neutron-anchor-json neutron_anchor_run_1a/best_candidate_neutron_anchor.json \
  --rank-by combined \
  --sqk-weight 0.35
```

## 6) Check that the neutron anchor was used

Open the proton summary:

```bash
cat proton_refine_run_1e/summary.txt
```

Look for:

```text
sqk_length_scale_source = neutron-anchor
```

If instead it says:

```text
sqk_length_scale_source = proton-fallback
```

then the neutron anchor was not picked up.
